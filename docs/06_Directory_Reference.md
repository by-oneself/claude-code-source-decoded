# 源码目录速查字典 (Directory Reference)

本文件提供对解包后的 `src` 文件夹下核心目录结构的鸟瞰。当你在开发或试图改动源程序的 `cli.js` 补丁时，可以通过此字典迅速定位你需要覆写的目标位置。

---

## 核心业务组件速查

### `src/components/` & `src/screens/`
- **作用**：控制终端所有的 UI 视图渲染（使用 React + Ink）。
- **重点文件**：
  - `Message.tsx`: 渲染大模型带 markdown 和行号代码气泡的聊天框。
  - `Spinner.tsx`: 提供多形态、平滑防止终端闪烁的加载动画圈。
  - `Chat.tsx` / `Auth.tsx`: 当系统需要进入长对话循环或引导身份登陆时拦截的总控 React 组件。

### `src/tools/`
- **作用**：模型拥有的所有“动作能力插件”。如果想修改大模型的行动权限或优化他的代码理解逻辑，来这里修改对应的 Tool。
- **重点目录**：
  - `BashTool/`: `exec` 控制层，包括超时切断与 PTY 挂载逻辑。
  - `FileEditTool/`: 包括智能 Diff 替换逻辑的实现。
  - `GlobTool/` & `GrepTool/`: 高速搜索代码段，防 token 爆炸。
  - `AgentTool/`: 实现嵌套代理逻辑与跨 Agent 上下文管理。
  - `WebFetchTool/`: 外网内容爬取与降维压缩。

### `src/coordinator/`
- **作用**：多代理调度的任务大总管。负责接收最初的指令，对任务做计划并下发给 Worker 子环境，协调最终结果并输出给 UI 层。
- **核心逻辑**：在这里理解一棵巨大复杂任务逻辑树是如何被分叉折叠解构的。

### `src/services/`
- **作用**：与端外交互的基础设施。
- **主要内容**：
  - `api/claude.ts`: 处理了所有连接 Anthropic 服务器的底层调用代码（Retry退避、重连等）。
  - `analytics/`: 所有埋点遥测事件收集与发包点。
  - `compact/compact.ts`: 负责大模型上下文长到触发雪崩前将无效聊天日志丢弃并重构出浓缩精华背景的核心折叠引擎。

### `src/utils/`
- **作用**：通用辅助大工具库集合。
- **重点内容**：
  - `messages.ts`: 构建发送给 AI 模型的格式化消息数组（System、Assistant、User 多角色的处理逻辑映射）。
  - `permissions/`: 处理高危命令拦截、免询问自动执行机制（附带核心 `yoloClassifier.ts` 纯本地高速危险判断拦截逻辑）。
  - `config.ts`: 获取机器的系统配置。

### `src/entrypoints/`
- **作用**：CLI 主程序运行入口。
- **核心**：`cli.tsx` 控制程序的降级适配、权限检查以及拉起主界面的 `QueryEngine` 主力循环通道。

## 修改实操小贴士
如果你要对系统做定制化修改（破除使用限额、修改模型人设或者替换其他第三方代理）：

1. **查找定位**：在本仓库使用诸如 `VS Code` 的全局搜索想要改的系统提示词或核心方法名。
2. **复制独门标记**：例如你想改 Agent 的行为，可以在这里找到特定的英文注释字符串或者特定的防飞（Caveat）标记标签（如 `<agent_warning>` 此类 XML 标签）。
3. **去本体修改**：打开你原版通过 npm 下载的全局包（通常在 C盘 或者 `/usr/local/lib/node_modules/@anthropic-ai/claude-code/`）。
4. **强行 Replace**：用编辑器打开那里面的那个巨大的单体文件 `cli.js`。利用刚才找到的独特标记字符串进行全局定位，顺藤摸瓜对那个混淆打包好的 JS 对象进行文本替换甚至生硬的语句覆盖。

这样由于你不必去忍受残缺依赖带来的重新编译（Re-bundle）地狱，你的魔改版必定能完美跑通！
